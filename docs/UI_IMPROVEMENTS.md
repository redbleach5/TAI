## Средний приоритет

- **Toasts** — анимация появления справа, кнопка закрытия, не перекрывать важные элементы.
- **Ресайз панелей** — тянущаяся граница между чатом и редактором, запоминание ширины (localStorage).
- **Ресайз сайдбара** — ширина файлового дерева настраивается перетаскиванием.
- **Тултипы** — везде, где только иконка (сайдбар, кнопки в чате), показывать подсказку при наведении.
- **Индикатор загрузки** — единый стиль «печатает…» (точки или скелетон), не дёргаться при стриминге.
- **Кнопка отправки** — визуально активна, когда есть текст (акцент), подсказка «Enter» / «Shift+Enter».

---

## Низкий приоритет / полировка

- **Тёмная/светлая тема** — переключатель в настройках, CSS-переменные для light.
- **Монохром в редакторе** — шрифт Fira Code / JetBrains Mono, одинаковый в чате и в редакторе.
- **Плавное появление сообщений** — лёгкий fade-in при добавлении сообщения в чат.
- **Командная палитра** — Cmd/Ctrl+K: быстрый поиск по действиям (новый чат, анализ, шаблоны).
- **Доступность** — aria-label у кнопок, навигация по списку чатов с клавиатуры (стрелки, Enter).
- **Мобильная версия** — чат на весь экран по тапу, сайдбар скрывается в бургер-меню.
